## 🚀 شروع کار

### پیش‌نیازها
- Node.js 18+ 
- npm یا yarn
- Supabase account

### نصب و راه‌اندازی

1. **کلون کردن پروژه**
```bash
git clone https://github.com/your-username/meditrackapp.git
cd meditrackapp
```

2. **نصب وابستگی‌ها**
```bash
npm install
```

3. **تنظیم متغیرهای محیطی**
```bash
cp env.example .env.local
```

4. **تنظیم Supabase**
- پروژه جدید در Supabase ایجاد کنید
- URL و API Key را در `.env.local` قرار دهید
- جداول مورد نیاز را ایجاد کنید

5. **اجرای پروژه**
```bash
npm run dev
```

6. **Build پروژه (برای deployment)**
```bash
npm run build
```

## 🚀 Deployment

### Vercel Deployment
پروژه آماده deployment روی Vercel است:

1. **اتصال به GitHub**
   - پروژه را به Vercel متصل کنید
   - تنظیمات محیطی را اضافه کنید

2. **متغیرهای محیطی**
```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
```

3. **Deploy**
   - Vercel به صورت خودکار build و deploy می‌کند
   - پروژه آماده استفاده خواهد بود
